## Creating a Claude AI Project for Daily Tracking

In this step, I am setting up claude.ai project so i can once write instructions which can be picked up automatically everytime we log a meal.

![Image](http://nextwork.ai/fulfilled_red_happy_silkie/uploads/ai-claude-notion-calorie_f8h2j6l4)

### Writing persistent instructions

A Claude.ai Project is a workspace that groups related conversations together. Persistent instructions help because every chat you start, the project automatically follows the instruction set.

![Image](http://nextwork.ai/fulfilled_red_happy_silkie/uploads/ai-claude-notion-calorie_a7c1e5g9)

### Testing the meal logging workflow

I tested my tracker by asking it to log a meal for me. Claude responded with first estimating macros and them logging them in my notion database. I verified it in Notion by opening up claude's entry.
